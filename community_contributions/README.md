# Community Contributions

This section of the repository contains variations of the code examples, case studies and notebooks
created by readers and shared with the world.

## Instructions

To make a contribution.

* Make a fork of the repository.
* Copy the code you want to contribute into this directory `community_contributions`
* Push it to your repository
* Issue a pull request to merge it into the `data-science-first` repository


## Community Guidelines

Please be considerate of others and refrain from using inflamatory or offensive language in your scripts or notebooks. You should expect that these code examples will be read by people of all ages and backgrounds.
